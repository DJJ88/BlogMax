Defining New Macros
RSS | Contents | A Day in the Life of a BlogMax Web Site

This page tells you how to define new BlogMax macros.

Basically, to define the {foo} macro, create a lisp function named weblog-macro-foo. Its argument list will be the same arguments required by the macro on your pages. Look at blogmax.el for some examples.
